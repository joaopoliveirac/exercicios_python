DESAFIOS:

1. Encontrar Elementos Únicos
Dada uma tupla com números repetidos, crie uma nova tupla apenas com os elementos únicos.

Exemplo:
Entrada: (1, 2, 2, 3, 4, 4, 5, 5, 5)
Saída: (1, 2, 3, 4, 5)

2. Jogo de Estatísticas com Tuplas
Peça ao usuário para digitar 10 números e armazene-os em uma tupla. Depois, exiba:

Quantos são pares

3. Verificar Anagrama com Tuplas
Peça ao usuário duas palavras e verifique se são anagramas usando tuplas.

Exemplo:
Entrada: "amor", "roma"
Saída: "São anagramas!"

4. Criar uma Tupla de Palavras Ordenadas
Peça ao usuário uma frase e armazene cada palavra em uma tupla ordenada alfabeticamente.

Exemplo:
Entrada: "Python é uma linguagem poderosa"
Saída: ('Python', 'é', 'linguagem', 'poderosa', 'uma')

5. Criar uma Tabela com Tuplas
